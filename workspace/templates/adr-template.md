## Template Usage Notes

1. **Number sequentially**: ADR-0001, ADR-0002, etc.
2. **Title format**: Short, descriptive, uses imperative mood ("Use JWT for auth" not "JWT authentication")
3. **Status flow**: Proposed → Accepted → (Deprecated or Superseded)
4. **Keep historical**: Never delete ADRs, mark as deprecated/superseded instead
5. **Link bidirectionally**: If ADR-0005 supersedes ADR-0002, update ADR-0002 to reference ADR-0005
